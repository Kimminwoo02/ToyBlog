# ToyBlog
학습한 내용들을 토대로 1인 프로젝트를 진행하면서 실무에 필요한 기술들을 익힘

## 개발 환경

* Intellij IDEA Ultimate
* Java 17
* Gradle 7.6.1
* Spring Boot 2.7.0

## 기술 세부 스택

* Spring Web
* Spring Data JPA
* Vue.js
* Axios
* ElementPlus
* Spring Security
* MySQL Driver
* Lombok
* Spring Boot DevTools
* Spring Configuration Processor

그 외

* QueryDSL
* Bootstrap
